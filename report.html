State 3 ways in which your site is accessible.

All my images have equivalent alternative text to allow anyone using a screen reader to have a description of the image. For users that can only use a keyboard I adjusted the focus indicator to be 3px thick and changed the colour to compliment the colour scheme of the website. I chose to have a contrasting background colour to the white text this helps people who are visually impared to read text on the page.

State 3 ways in which you have considered usability in your site.

As people commonly read a web page as they woulda piece of writing from top left across to the right then down, i placed the title and the home page link of the website in the top left. This allows the user to instantly be able to know what site they are on. Just below the title is the menu bar. this as the second thing that the user will register allows them to instantly navigate the site. I also included a second menu at the bottom of each page with links to the rest of the site; this allows the user to navigate the site after getting to the end of a page without having to scroll back to the top of the page.

Sudoku

Sudoku is an application I created for the module Problem Solving for Computer Science. The application generates a four by four sudoku grid then removes a defined number of elements within the grid. The application also has the functionality of solving the sudoku puzzle. in the image to the left you can see two examples of this. the first two images are sudoku puzzles that have been generated the first with 8 missing elements the second with 10. The second two images are the same as the first two sudoku puzzles after they have been solved.

The two files can be run using Node.js in a command-line interface (CLI).
